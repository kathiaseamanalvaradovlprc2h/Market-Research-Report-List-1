<p><p>Time-Sensitive Networking (TSN) is a set of IEEE standards that enable deterministic real-time communication over Ethernet networks. The market types for TSN include IEEE 802.1 AS for time synchronization, IEEE 802.1 Qca for stream reservation, IEEE 802.1 AS-Rev for resource management, IEEE 802.1 Qbv for traffic shaping, IEEE 802.1 Qci for path control, IEEE 802.1 CB for frame replication and elimination, IEEE 802.1 Qcc for stream identification, IEEE 802.1 Qch for cyclic transmission, and IEEE 802.1 CM for configuration and management.</p></p>
