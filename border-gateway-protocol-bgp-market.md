<p><ul><li>Interior Border Gateway Protocol (IBGP)</li><li>Exterior Border Gateway Protocol (EBGP)</li></ul></p>
<p><p>Border Gateway Protocol (BGP) is a routing protocol used to facilitate communication between different autonomous systems (AS) on the internet. BGP market types include Interior Border Gateway Protocol (IBGP), which is used within a single AS to exchange routing information, and Exterior Border Gateway Protocol (EBGP), which is used to exchange routing information between different AS. IBGP is used to maintain internal routing within a single organization, while EBGP is used for routing between different organizations or ISPs.</p></p>
